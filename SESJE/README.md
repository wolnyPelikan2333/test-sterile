# SESJE — system pracy

## Co to jest
Ten katalog zawiera JEDEN aktywny plik stanu pracy:
- `AKTYWNA.md` — aktualny stan (TERAZ / W TOKU / POTEM / KONTEKST)

Plik opisuje STAN, nie plan. Może się zmieniać w ciągu dnia.

## Jak z tego korzystać
- start dnia / sesji: czytaj `AKTYWNA.md`
- koniec dnia: aktualizuj stan (przesuwaj rzeczy między sekcjami)
- mikrokroki i daty są dozwolone (daty = przypominajki)

## Kiedy zamykać AKTYWNA.md
Zamknij plik, gdy:
- minęło kilka dni
- zmienił się kontekst
- plik zaczyna ciążyć

Nie trzeba kończyć wszystkich zadań.

## Archiwum
- stare pliki stanu trafiają do `ARCHIWUM/`
- archiwalne pliki nie są poprawiane

## Bezpieczeństwo
Jeśli pojawia się chaos lub przeciążenie:
- STOP
- zapisz aktualny stan
- nie zamykaj pliku
