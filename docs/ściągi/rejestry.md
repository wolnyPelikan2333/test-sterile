================ REJESTRY =================

y / d / p        → rejestr domyślny "
"ayy             → kopiuj linię do rejestru a
"ap              → wklej z rejestru a

"+y              → kopiuj do systemu (Ctrl+C)
"+p              → wklej z systemu (Ctrl+V)

"0p              → wklej OSTATNIE kopiowanie (ratunek!)
"_dd             → usuń BEZ psucia schowka

:reg             → pokaż wszystkie rejestry


================ MAKRA ====================

qa               → zacznij nagrywać makro do rejestru a
q                → zakończ nagrywanie

@a               → odtwórz makro a
@@               → powtórz ostatnie makro

10@a             → wykonaj makro a 10 razy

Uwaga:
- makra SIEDZĄ w rejestrach (a–z)
- można je wkleić jak tekst: "ap
